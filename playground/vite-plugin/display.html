🟦 Basic Display Values

Block vs Inline vs Inline-Block

block - expands to full width
inline - content width inline - side by side
inline-block inline-block

HTML

<div class="block">block - expands to full width</div>
<span class="inline">inline - content width</span>
<span class="inline">inline - side by side</span>
<span class="inline-block">inline-block</span>
<span class="inline-block">inline-block</span>

Generated CSS

.block { display: block; }
.inline { display: inline; }
.inline-block { display: inline flow-root; }

📋 Available Display Classes List

Basic Display

  • block → display: block;
  • inline → display: inline;
  • inline-block → display: inline flow-root;
  • none → display: none;
  • hidden → display: none;
  • contents → display: contents;

Layout

  • flex → display: block flex;
  • inline-flex → display: inline flex;
  • grid → display: block grid;
  • inline-grid → display: inline grid;
  • flow-root → display: block flow-root;

Table Types

  • table → display: block table;
  • inline-table → display: inline table;
  • table-cell → display: table-cell;
  • table-row → display: table-row;
  • table-caption → display: table-caption;
  • list-item → display: block flow list-item;
