📏 基本的なフォントサイズ

フィボナッチ数列ベースのサイズ階層

font-size-xs (8px) - 最小サイズ
font-size-sm (12px) - 小サイズ
font-size-md (20px) - 中サイズ (デフォルト)
font-size-lg (32px) - 大サイズ
font-size-xl (52px) - 特大サイズ

生成されるCSS

.font-size-xs { font-size: 8px; }
.font-size-sm { font-size: 12px; }
.font-size-md { font-size: 20px; }
.font-size-lg { font-size: 32px; }
.font-size-xl { font-size: 52px; }

🎯 カスタム値

任意の値でのサイズ指定

font-size-[14px] - ピクセル指定
font-size-[1.2em] - em単位
font-size-[1.5rem] - rem単位
font-size-[calc(1rem+5px)] - calc関数
font-size-[clamp(12px,2vw,24px)] - レスポンシブサイズ

HTML

<div class="font-size-[14px]">14px</div>
<div class="font-size-[1.2em]">1.2em</div>
<div class="font-size-[1.5rem]">1.5rem</div>
<div class="font-size-[calc(1rem+5px)]">calc</div>
<div class="font-size-[clamp(12px,2vw,24px)]">clamp</div>

生成されるCSS

.font-size-\[14px\] { font-size: 14px; }
.font-size-\[1\.2em\] { font-size: 1.2em; }
.font-size-\[1\.5rem\] { font-size: 1.5rem; }
.font-size-\[calc\(1rem\+5px\)\] { font-size: calc(1rem + 5px); }
.font-size-\[clamp\(12px\,2vw\,24px\)\] { font-size: clamp(12px, 2vw, 24px); }

📋 利用可能なFont Sizeクラス一覧

デフォルトサイズ

  • font-size-xs → 8px
  • font-size-sm → 12px
  • font-size-md → 20px
  • font-size-lg → 32px
  • font-size-xl → 52px

フィボナッチ数列:
8, 12(8+4), 20(12+8), 32(20+12), 52(32+20)

カスタム値

  • font-size-[14px] → 14px
  • font-size-[1.2em] → 1.2em
  • font-size-[1.5rem] → 1.5rem
  • font-size-[16pt] → 16pt
  • font-size-[2vw] → 2vw
  • font-size-[2vh] → 2vh

CSS関数

  • font-size-[calc(1rem+5px)]
  • font-size-[min(20px,4vw)]
  • font-size-[max(14px,1rem)]
  • font-size-[clamp(12px,2vw,24px)]
  • font-size-[var(--font-size)]

推奨レスポンシブ:
clamp(最小, 可変, 最大) を使用
