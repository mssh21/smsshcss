📘 Subgridとは？

Subgridは、子グリッドコンテナが親のグリッドライン構造を継承する機能です。 これにより、ネストしたレイアウトでも親グリッドとの整合性を保てます。

利用可能なSubgridクラス

/* 列方向のサブグリッド */
.subgrid-cols {
  display: subgrid;
  grid-template-columns: subgrid;
}

/* 行方向のサブグリッド */
.subgrid-rows {
  display: subgrid;
  grid-template-rows: subgrid;
}

/* 両方向のサブグリッド */
.subgrid-both {
  display: subgrid;
  grid-template-columns: subgrid;
  grid-template-rows: subgrid;
}

⚠️ ブラウザ対応状況： Subgridは比較的新しい機能です。Firefox 71+、Safari 16+、Chrome 117+ で対応。 古いブラウザでは通常のグリッドとして動作します。

📋 利用可能なGridクラス一覧

Template

  • grid-cols-1 〜 grid-cols-12
  • grid-rows-1 〜 grid-rows-6
  • grid-cols-none → none;
  • grid-rows-none → none;

カスタム値:

  • grid-cols-[5] → repeat(5, minmax(0, 1fr));
  • grid-rows-[4] → repeat(4, minmax(0, 1fr));

Span

  • col-span-1 〜 col-span-12
  • row-span-1 〜 row-span-6
  • col-span-full → span all;
  • row-span-full → span all;

カスタム値:

  • col-span-[5] → span 5 / span 5;
  • row-span-[3] → span 3 / span 3;

Position

  • col-start-1 〜 col-start-13
  • col-end-1 〜 col-end-13
  • row-start-1 〜 row-start-7
  • row-end-1 〜 row-end-7
  • col-start-auto → auto;
  • row-start-auto → auto;

カスタム値:

  • col-start-[2] → grid-column-start: 2;
  • row-end-[5] → grid-row-end: 5;
