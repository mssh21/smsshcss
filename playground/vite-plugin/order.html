🔢 デフォルトOrderクラス (Flexbox)

基本的なorder値 (1-12)

order-3 (3番目)
order-1 (1番目)
order-5 (5番目)
order-2 (2番目)
order-4 (4番目)

HTML

<div class="flex gap-[10px]">
  <div class="order-3">order-3 (3番目)</div>
  <div class="order-1">order-1 (1番目)</div>
  <div class="order-5">order-5 (5番目)</div>
  <div class="order-2">order-2 (2番目)</div>
  <div class="order-4">order-4 (4番目)</div>
</div>

生成されるCSS

.order-1 { order: 1; }
.order-2 { order: 2; }
.order-3 { order: 3; }
.order-4 { order: 4; }
.order-5 { order: 5; }

⚡ 特別なOrder値

order-first, order-last, order-none

order-last (最後)
order-none (元順序)
order-first (最初)
order無し (元順序)

HTML

<div class="flex gap-[10px]">
  <div class="order-last">order-last (最後)</div>
  <div class="order-none">order-none (元順序)</div>
  <div class="order-first">order-first (最初)</div>
  <div>order無し (元順序)</div>
</div>

生成されるCSS

.order-first { order: -9999; }
.order-last { order: 9999; }
.order-none { order: 0; }

🎯 カスタム値テスト

任意のorder値

order-[100]
order-[-1]
order-[50]
order-[13]
order-[0]

HTML

<div class="flex gap-[10px]">
  <div class="order-[100]">order-[100]</div>
  <div class="order-[-1]">order-[-1]</div>
  <div class="order-[50]">order-[50]</div>
  <div class="order-[13]">order-[13]</div>
  <div class="order-[0]">order-[0]</div>
</div>

生成されるCSS

.order-\[100\] { order: 100; }
.order-\[-1\] { order: -1; }
.order-\[50\] { order: 50; }
.order-\[13\] { order: 13; }
.order-\[0\] { order: 0; }

CSS変数とcalc()関数

order-[var(--custom-order)]
order-2
order-[calc(10+10)]
order-1

CSS変数とcalc()の例

<div style="--custom-order: 25;">
  <div class="order-[var(--custom-order)]">CSS変数</div>
  <div class="order-[calc(10+10)]">calc関数</div>
</div>

📋 利用可能なOrderクラス一覧

基本Order値

  • order-1 → order: 1;
  • order-2 → order: 2;
  • order-3 → order: 3;
  • order-4 → order: 4;
  • order-5 → order: 5;
  • order-6 → order: 6;
  • order-7 → order: 7;
  • order-8 → order: 8;
  • order-9 → order: 9;
  • order-10 → order: 10;
  • order-11 → order: 11;
  • order-12 → order: 12;

特別なOrder値

  • order-first → order: -9999;
  • order-last → order: 9999;
  • order-none → order: 0;

カスタム値

  • order-[13] → order: 13;
  • order-[-1] → order: -1;
  • order-[100] → order: 100;
  • order-[var(--order)] → order: var(--order);
  • order-[calc(10+5)] → order: calc(10+5);
