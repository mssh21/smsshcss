📦 基本的なOverflow制御

4つの基本動作

overflow-auto

この内容は親要素より長いテキストです。必要に応じて自動的にスクロールバーが表示されます。 Lorem ipsum dolor sit amet consectetur adipisicing elit.

overflow-hidden

この内容は親要素より長いテキストです。はみ出た部分は隠されます。 Lorem ipsum dolor sit amet consectetur adipisicing elit.

overflow-scroll

この内容は親要素より長いテキストです。常にスクロールバーが表示されます。 Lorem ipsum dolor sit amet consectetur.

overflow-visible

この内容は親要素より長いテキストです。はみ出た部分も表示されます。 Lorem ipsum dolor sit amet consectetur adipisicing elit.

生成されるCSS

.overflow-auto { overflow: auto; }
.overflow-hidden { overflow: hidden; }
.overflow-scroll { overflow: scroll; }
.overflow-visible { overflow: visible; }

✂️ Clip制御

overflow-clip の使用例

overflow-clip

overflow-clipは、はみ出た部分をクリップ（切り取り）します。 hiddenとよく似ていますが、スクロール機構を完全に無効化します。

overflow-x-clip

横方向のクリップ制御

生成されるCSS

.overflow-clip { overflow: clip; }
.overflow-x-clip { overflow-x: clip; }
.overflow-y-clip { overflow-y: clip; }

🎪 実用的な例

よくある使用場面

カードコンテンツ

カードタイトル

長いテキストコンテンツがここに入ります。カードの高さを固定して、はみ出た部分は隠しています。 これにより統一感のあるレイアウトが作れます。

タブレット形式

タブ 1
タブ 2

タブ 4
タブ 5

タブのコンテンツエリア。縦にスクロール可能です。

実用例のCSS

/* カード例 */
.card-content { overflow: hidden; }

/* タブ例 */
.tab-list { overflow-x: auto; }
.tab-content { overflow-y: auto; }

📋 利用可能なOverflowクラス一覧

基本Overflow

  • overflow-auto → overflow: auto;
  • overflow-hidden → overflow: hidden;
  • overflow-visible → overflow: visible;
  • overflow-scroll → overflow: scroll;
  • overflow-clip → overflow: clip;

横方向制御

  • overflow-x-auto → overflow-x: auto;
  • overflow-x-hidden → overflow-x: hidden;
  • overflow-x-visible → overflow-x: visible;
  • overflow-x-scroll → overflow-x: scroll;
  • overflow-x-clip → overflow-x: clip;

縦方向制御

  • overflow-y-auto → overflow-y: auto;
  • overflow-y-hidden → overflow-y: hidden;
  • overflow-y-visible → overflow-y: visible;
  • overflow-y-scroll → overflow-y: scroll;
  • overflow-y-clip → overflow-y: clip;
