📋 利用可能なZ-Indexクラス一覧

デフォルト値

  • z-0 → z-index: 0;
  • z-10 → z-index: 10;
  • z-20 → z-index: 20;
  • z-30 → z-index: 30;
  • z-40 → z-index: 40;
  • z-50 → z-index: 50;
  • z-auto → z-index: auto;

カスタム値

  • z-[1] → z-index: 1;
  • z-[100] → z-index: 100;
  • z-[999] → z-index: 999;
  • z-[9999] → z-index: 9999;
  • z-[-1] → z-index: -1;
  • z-[calc(10+5)] → z-index: calc(10+5);
  • z-[var(--z)] → z-index: var(--z);

よくある値

  • 背景・基本: z-0, z-10
  • コンテンツ: z-20, z-30
  • オーバーレイ: z-40
  • モーダル: z-50
  • ツールチップ: z-[100]
  • 最上位: z-[9999]
