📋 Available Z-Index Classes List

Default Values

  • z-0 → z-index: 0;
  • z-10 → z-index: 10;
  • z-20 → z-index: 20;
  • z-30 → z-index: 30;
  • z-40 → z-index: 40;
  • z-50 → z-index: 50;
  • z-auto → z-index: auto;

Custom Values

  • z-[1] → z-index: 1;
  • z-[100] → z-index: 100;
  • z-[999] → z-index: 999;
  • z-[9999] → z-index: 9999;
  • z-[-1] → z-index: -1;
  • z-[calc(10+5)] → z-index: calc(10+5);
  • z-[var(--z)] → z-index: var(--z);

Common Values

  • Background/Basic: z-0, z-10
  • Content: z-20, z-30
  • Overlay: z-40
  • Modal: z-50
  • Tooltip: z-[100]
  • Top level: z-[9999]
